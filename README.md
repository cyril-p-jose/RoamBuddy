# 🧭 About RoamBuddy

**RoamBuddy** is a travel-planning web interface designed to help users explore destinations and plan trips easily.

The project focuses on a **simple, modern UI** where users can discover travel locations and organize their journeys in a visually appealing way.

It is built as a **frontend project** to practice modern web design and interface development.

---

# 🚀 Features

✨ Clean and modern travel website design
🗺️ Destination exploration interface
📱 Responsive layout for different screens
⚡ Fast and lightweight static webpage
